## Card Components: Layout Contract

### Primary вҖ” TicketDestinationCard

**Shape:** `TicketShape` вҖ” rounded rect (cornerRadius: 28pt) with circular notches punched from each side at the stub seam. Notch radius: 11pt, notch position: 66pt + 11pt from bottom.

**Structure (top to bottom):**

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ  ScenicArtBand (aspectRatio 1.5)в”Ӯ  вҶҗ category-driven procedural art
в”Ӯ  [PostageStamp top-trailing]    в”Ӯ  вҶҗ 44Г—52pt, 4pt border-dash, rotated 4В°
в”Ӯ  [Scrim overlay bottom-leading] в”Ӯ  вҶҗ eyebrow (11pt semibold, tracking 1.4)
в”Ӯ    + POI name (28pt semibold)   в”Ӯ
в”Ӯ    + Vibe line (14pt, optional) в”Ӯ
в”ң В·  В·  В·  В·  В·  В·  В·  В·  В· в”Җв”Җв”Җв”Җв”Ө  вҶҗ DashedRule (1.5pt, dash [5,5]), 22pt zone
в”Ӯ  STUB: STATUS     ON FOOT       в”Ӯ  вҶҗ 66pt height, horizontal padding Theme.Spacing.l
в”Ӯ  [openText]       [walkText?]   в”Ӯ  вҶҗ 15pt semibold, label 9pt heavy
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

**Shadow:** `radius: 16, y: 8, opacity: 0.12` (black).
**Stroke:** `Theme.Palette.cardBorder`, 1pt, same `TicketShape`.
**Fill:** `Theme.Palette.cardSurface`.

**Binding fields from AnchorViewModel:**
- `poi.name` вҶ’ `name`
- `"\(poi.category.label) В· \(poi.neighborhood)"` вҶ’ `eyebrow`
- `viewModel.vibeLine` вҶ’ `description` (optional; shown in header scrim when non-nil)
- `poi.poiRef` + `poi.category` + `viewModel.context?.weather ?? .clear` вҶ’ drives `ScenicArtBand`
- `viewModel.pickIsLikelyOpen ? "Open Now" : "Hours vary"` вҶ’ `openText`
- `viewModel.walkInfo.map { "\($0.walkMinutes) min" }` вҶ’ `walkText` (optional; stub "ON FOOT" column hidden when nil)

**Accessibility:** `.accessibilityElement(children: .combine)` + `.accessibilityLabel("\(name), \(eyebrow)")`.

| State | Component | Trigger |

## Interaction Contract

### Pinned Action Area

Shown only when `viewModel.pick != nil && !viewModel.isLoading && viewModel.arrivalResult == nil`.

**"Take me there" (primary CTA):**
- Style: `.buttonStyle(.glassProminent)`, tint `Theme.Garden.moss`, `.controlSize(.large)`
- Label: `Label("Take me there", systemImage: "figure.walk")`
- Font: `Theme.Typography.body(17, weight: .semibold)`
- Action: `viewModel.openInMaps()`
- Width: `maxWidth: .infinity`, vertical padding 6pt

**"Another" (re-roll):**
- Style: `.buttonStyle(.glass)`, tint `Theme.Garden.mossLight`
- Label: `Label("Another", systemImage: "arrow.clockwise")`
- Font: `Theme.Typography.body(15, weight: .medium)`
- Action: `withAnimation(.smooth) { viewModel.rollAnother() }`
- Width: `maxWidth: .infinity` within HStack, vertical padding 4pt

**"I'm here" (arrival):**
- Style: `.buttonStyle(.glass)`, tint `Theme.Garden.pine`
- Label: `Label("I'm here", systemImage: "mappin.and.ellipse")`
- Font: `Theme.Typography.body(15, weight: .medium)`
- Action: `Task { await viewModel.arrive() }`
- Disabled: `!viewModel.arrivalProximity.allowsCheckIn`
- Width: `maxWidth: .infinity` within HStack, vertical padding 4pt

**Status note (above actions):**
- `arrivalBlocked == true` вҶ’ icon `location.slash.fill` + "Not there yet вҖ” get a little closer to check in."
- `arrivalHint != nil` вҶ’ icon `location.circle` + hint string
- Font: `Theme.Typography.body(13, weight: .medium)`, color `Theme.Palette.secondary`
- Animation: `.smooth` on `arrivalBlocked` and `arrivalHint` changes

**Action container:** `GlassEffectContainer(spacing: Theme.Spacing.m)` wrapping `VStack(spacing: Theme.Spacing.m)`.

**Bottom fade background:** `LinearGradient` from `Color(hex: "FBF2E0").opacity(0)` вҶ’ `.opacity(0.92)` вҶ’ `.opacity(1.0)`, height matched to action area, `allowsHitTesting(false)`.

### Card Transition on Re-Roll

`TicketDestinationCard` must have `.id(viewModel.pick?.poiRef)` so SwiftUI replaces the view identity on re-roll. Transition: `.opacity.combined(with: .scale(scale: 0.97))`. Animation: `.smooth`.

### Reward Overlay

Already wired in `AnchorView`. Not changed by Phase 1. Triggered by `.onChange(of: viewModel.arrivalResult)` when `tiersGained > 0`.
